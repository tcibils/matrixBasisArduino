General code for arduino led matrix 6x10 matrix as I coded it =)


The goal of this branch is to be a reference point for the controller code for SNES, with my matrix

*Links are as below :*

Black = ground

Green = +5V (wtf, I know)

Yellow = Clock

White = Latch

Red = Data
